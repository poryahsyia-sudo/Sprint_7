Тесты API Яндекс Самокат
Описание

Автоматические тесты для проверки работы API учебного сервиса Яндекс Самокат.
Покрываются проверки создания и авторизации курьера, создания заказа и получения списка заказов.

Используемые технологии

Python 3
Pytest
Requests

Проверяемые ручки API
POST /api/v1/courier — создание курьера
POST /api/v1/courier/login — авторизация курьера
POST /api/v1/orders — создание заказа
GET /api/v1/orders — список заказов
